# VLESS Helm Chart

Данный Helm-чарт предназначен для развертывания прокси-сервера на основе протокола VLESS (Xray/V2Fly) в кластере Kubernetes.

## Особенности

- Поддержка протокола VLESS (среда исполнения Xray).
- Автоматическая генерация или ручное указание UUID (ID клиента).
- Поддержка TLS (termination на уровне Ingress или внутри пода).

## Требования

- Kubernetes 1.19+
- Helm 3.2.0+
- (Опционально) Ingress Controller для проброса трафика через TLS.
